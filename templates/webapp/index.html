Phase 1: BRPG

The RPGCAT project is currently in phase 1; the creation of the game entitled "Basic Role Playing Game" (BRPG)

BRPG is to be the skeleton upon which all other projects are built and the feature list outline of it can be found on the project github site located here

In short, however, it aims to provide the following basic functionality, which will be built on in the future:

  • Log in + char creation
  • Move around map
  • Basic player interactions
  • Basic items
  • Basic skills
  • Basic NPC/Quest Interaction

Once we have implemented the components of BRPG we will move onto phase 2; to be disclosed at a later date.
